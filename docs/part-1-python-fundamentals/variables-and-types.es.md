# Variables y Tipos

## Que aprenderas

- Bases del tipado dinamico
- Tipos integrados comunes
- Patrones de conversion de tipos

## Explicacion

Las variables en Python son nombres enlazados a objetos. Un mismo nombre
puede referirse despues a otro tipo, pero la claridad mejora si mantienes un uso
de tipos predecible.

## Ejemplo

Ejecucionalo en el navegador:

<div class="py-run" markdown="1">

```python
count = 3
price = 19.99
title = "Python Book"
print(type(count), type(price), type(title))
```

</div>

## Errores comunes

- Confundir `=` (asignacion) con `==` (comparacion)
- Olvidar que `input()` devuelve texto
