# Primeros Pasos

## Que aprenderas

- Como ejecutar codigo Python
- Script vs modo interactivo
- Estructura basica de un programa

## Explicacion

Python puede usarse en un REPL interactivo o ejecutando archivos `.py`.
Usa el REPL para experimentar rapido y scripts para programas reutilizables.

## Ejemplo

Ejecucionalo en el navegador:

<div class="py-run" markdown="1">

```python
name = "Ada"
print(f"Hello, {name}!")
```

</div>

## Errores comunes

- Ejecutar codigo con una version de Python distinta a la esperada
- Mezclar tabs y espacios en la indentacion
