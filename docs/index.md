# Python Book

Learn Python through structured chapters, examples, and exercises.

## What this project is

- Chapter-first learning path (like a handbook)
- Clean sidebar navigation with next/previous chapter flow
- Search-first reading experience
- Exercises separated for reuse and future interactive modes
- In-browser Python execution on selected popular chapters via Pyodide

## Learning map

1. Part 1: Python fundamentals (starts with a 3-4 month beginner roadmap)
2. Part 2: Standard library essentials
3. Part 3: Real world Python
