# Python Book

Aprende Python con capitulos estructurados, ejemplos y ejercicios.

## Que es este proyecto

- Ruta de aprendizaje centrada en capitulos (como un manual)
- Navegacion clara en barra lateral con flujo de capitulo siguiente/anterior
- Experiencia de lectura centrada en busqueda
- Ejercicios separados para reutilizacion y modos interactivos futuros
- Ejecucion de Python en el navegador en capitulos populares con Pyodide

## Mapa de aprendizaje

1. Parte 1: Fundamentos de Python (empieza con una ruta para principiantes de 3-4 meses)
2. Parte 2: Esenciales de la libreria estandar
3. Parte 3: Python en el mundo real

## Ultima actualizacion

Las fechas de actualizacion por pagina basadas en Git estan disponibles cuando
el plugin `git-revision-date-localized` esta habilitado.
